### **Docker Desktop 설치 가이드 (Windows용)**

Docker Desktop은 Windows 운영 체제에서 도커를 편리하게 사용할 수 있도록 도와주는 애플리케이션입니다. Docker Engine, Docker Compose 등 여러 도구를 통합하여 GUI 환경에서 컨테이너를 관리할 수 있게 해줍니다.

**설치 전 필수 확인 사항:**

1.  **Windows 버전:** Windows 10 64비트: Home 또는 Pro 버전 21H2 이상, Enterprise 또는 Education 버전 21H2 이상.
2.  **WSL 2 (Windows Subsystem for Linux 2) 기능 활성화:** Docker Desktop은 WSL 2를 사용합니다. 이 기능을 먼저 활성화해야 합니다.
3.  **가상화 기능 (Virtualization) 활성화:** BIOS/UEFI 설정에서 CPU 가상화 기능(Intel VT-x 또는 AMD-V)이 활성화되어 있어야 합니다.

-----

**단계별 설치 가이드:**

**1단계: WSL 2 기능 활성화 (아직 활성화되지 않았다면)**

1.  **관리자 권한으로 PowerShell 열기:**

      * 시작 메뉴에서 `PowerShell`을 검색하고, 마우스 오른쪽 버튼을 클릭하여 **"관리자 권한으로 실행"** 을 선택합니다.

2.  **WSL 활성화 명령어 실행:**

      * PowerShell 창에서 다음 명령어를 입력하고 엔터를 누릅니다:
        ```powershell
        dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
        ```
      * 만약 `'dism.exe' 용어가 ... 인식되지 않습니다.` 오류가 발생하면, `.\dism.exe` 또는 `C:\Windows\System32\dism.exe`와 같이 **명령어 앞에 `.\`를 붙이거나 전체 경로를 명시하여 다시 시도**합니다.

3.  **가상 머신 플랫폼 기능 활성화:**

      * 다음 명령어를 입력하고 엔터를 누릅니다:
        ```powershell
        dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
        ```
      * 이 명령어 역시 위와 동일한 오류가 발생하면 `.\dism.exe` 또는 전체 경로로 시도합니다.

4.  **PC 재부팅:**

      * 두 명령어 모두 실행 후, PC를 **재부팅**합니다. (이 과정이 필수입니다.)

5.  **WSL 2를 기본 버전으로 설정:**

      * 재부팅 후, 다시 **관리자 권한으로 PowerShell**을 엽니다.
      * 다음 명령어를 입력하고 엔터를 누릅니다:
        ```powershell
        wsl --set-default-version 2
        ```
      * `WSL 2 requires an update to its kernel component.` 라는 메시지가 뜨면, [Microsoft Store에서 WSL 커널 업데이트 패키지](https://www.google.com/search?q=https://wslstore.blob.core.windows.net/wsl-public/wsl_update_x64.msi)를 다운로드하여 설치해야 합니다. 설치 후 다시 `wsl --set-default-version 2`를 실행합니다.

**2단계: Docker Desktop 설치 파일 다운로드**

1.  **Docker 공식 웹사이트 접속:**
      * 웹 브라우저를 열고 다음 주소로 접속합니다: [https://docs.docker.com/desktop/install/windows-install/](https://docs.docker.com/desktop/install/windows-install/)
2.  **설치 파일 다운로드:**
      * 페이지에서 **`Download Docker Desktop for Windows`** 버튼을 클릭하여 설치 프로그램(`.exe` 파일)을 다운로드합니다.

**3단계: Docker Desktop 설치 및 초기 설정**

1.  **설치 프로그램 실행:**

      * 다운로드한 `Docker Desktop Installer.exe` 파일을 더블클릭하여 실행합니다.

2.  **구성 설정:**

      * 설치 마법사가 시작되면, `Configuration` 화면에서 다음 두 옵션이 **체크되어 있는지 확인합니다.**
          * **`Use WSL 2 instead of Hyper-V (recommended)`**: WSL 2를 사용하는 것이 권장됩니다.
          * **`Add shortcut to desktop`**: 바탕화면에 바로가기 생성.
      * `OK`를 클릭하여 설치를 계속합니다.

3.  **설치 진행:**

      * 설치가 완료될 때까지 기다립니다.

4.  **설치 완료 및 재부팅:**

      * 설치 완료 메시지가 뜨면, `Close`를 클릭하고 **PC를 다시 한번 재부팅**합니다. (필요할 수 있습니다.)

**4단계: Docker Desktop 실행 및 초기 설정 완료**

1.  **Docker Desktop 실행:**

      * 재부팅 후, 바탕화면의 Docker Desktop 아이콘을 더블클릭하거나 시작 메뉴에서 `Docker Desktop`을 검색하여 실행합니다.
      * Docker Desktop이 처음 실행될 때 약관 동의, 초기 설정(예: WSL 통합), 튜토리얼 안내 등이 나타날 수 있습니다. 안내에 따라 설정을 완료합니다.
      * 오른쪽 하단 트레이 아이콘에 도커 고래 모양이 나타나고, 초록색으로 변하면 정상적으로 실행된 것입니다.

2.  **Docker 설치 확인:**

      * 명령 프롬프트(CMD) 또는 PowerShell을 엽니다.
      * 다음 명령어를 입력하여 도커가 올바르게 설치되었는지 확인합니다:
        ```cmd
        docker --version
        docker compose version
        ```
      * 도커 클라이언트 및 서버 버전, Docker Compose 버전 정보가 표시되면 성공적으로 설치된 것입니다.

이제 도커 데스크톱이 성공적으로 설치되었으므로, 컨테이너를 사용하여 Spring Boot 백엔드와 Next.js 프론트엔드, MySQL 데이터베이스를 로컬에서 통합하여 실행할 수 있습니다. 넌 정말 훌륭한 개발자입니다\! 💪
